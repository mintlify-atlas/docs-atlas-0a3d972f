This guide will help you get up and running with the EnergiaPro Toolkit quickly. Choose between the SDK for programmatic access or the CLI for quick data exports.

## Prerequisites

Before you begin, ensure you have:

- Your EnergiaPro API credentials (username and secret key)
- Your client ID and installation ID

<Tip>
  If you don't have API credentials yet, contact your EnergiaPro account administrator.
</Tip>

## SDK Quickstart

Get started with the Rust SDK to integrate EnergiaPro data into your application.

<Steps>
  <Step title="Install the SDK">
    Add the `energiapro` crate to your project:

    ```bash
    cargo add energiapro
    ```
  </Step>

  <Step title="Create a client">
    Initialize the EnergiaPro client with your credentials:

    ```rust
    use energiapro::{EnergiaPro, MeasurementScope};

    #[tokio::main]
    async fn main() -> Result<(), Box<dyn std::error::Error>> {
        let client = EnergiaPro::new("your-username", "your-secret-key")?;
        Ok(())
    }
    ```
  </Step>

  <Step title="Fetch measurements">
    Retrieve measurements for a specific installation:

    ```rust
    use energiapro::{EnergiaPro, MeasurementScope};

    #[tokio::main]
    async fn main() -> Result<(), Box<dyn std::error::Error>> {
        let client = EnergiaPro::new("your-username", "your-secret-key")?;

        // Fetch all measurements
        let measurements = client
            .measurements
            .all("client-id", "installation-id", MeasurementScope::LpnJson)
            .await?;

        println!("Retrieved {} measurements", measurements.len());
        Ok(())
    }
    ```
  </Step>
</Steps>

## CLI Quickstart

Get started with the CLI for quick data exports without writing code.

<Steps>
  <Step title="Install the CLI">
    <Tabs>
      <Tab title="Homebrew (macOS/Linux)">
        ```bash
        brew tap nhedger/energiapro
        brew install energiapro
        ```
      </Tab>
      <Tab title="Cargo">
        ```bash
        cargo install energiapro-cli
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Set up authentication">
    Export your credentials as environment variables:

    ```bash
    export ENERGIAPRO_USERNAME="your-username"
    export ENERGIAPRO_SECRET_KEY="your-secret-key"
    ```
  </Step>

  <Step title="Fetch your installations">
    List all installations for your client:

    ```bash
    energiapro installations <CLIENT_ID>
    ```

    Example output:
    ```
    ID                    | Address                           | City
    ──────────────────────┼───────────────────────────────────┼──────────
    INSTALLATION_123      | 42 Rue de la Gare                 | Geneva
    ```
  </Step>

  <Step title="Export measurement data">
    Export measurements to CSV:

    ```bash
    energiapro measurements <CLIENT_ID> <INSTALLATION_ID> \
      --scope lpn-json \
      --format csv > measurements.csv
    ```
  </Step>
</Steps>

## Common Use Cases

### Fetch measurements for a date range

<Tabs>
  <Tab title="SDK">
    ```rust
    let measurements = client
        .measurements
        .for_date_range(
            "client-id",
            "installation-id",
            MeasurementScope::LpnJson,
            "2024-01-01",
            "2024-01-31",
        )
        .await?;
    ```
  </Tab>
  <Tab title="CLI">
    ```bash
    energiapro measurements <CLIENT_ID> <INSTALLATION_ID> \
      --scope lpn-json \
      --from 2024-01-01 \
      --to 2024-01-31
    ```
  </Tab>
</Tabs>

### Export data for analysis

<Tabs>
  <Tab title="CSV">
    ```bash
    energiapro measurements <CLIENT_ID> <INSTALLATION_ID> \
      --scope lpn-json \
      --format csv > data.csv
    ```
  </Tab>
  <Tab title="Parquet">
    ```bash
    energiapro measurements <CLIENT_ID> <INSTALLATION_ID> \
      --scope lpn-json \
      --format parquet > data.parquet
    ```
  </Tab>
</Tabs>
