## Overview

The `MeasurementsResource` provides methods for retrieving measurement data from the EnergiaPro API. Access it through the `measurements` field on the `EnergiaPro` client.

## Methods

### get

Retrieve measurements for a given installation and optional date range.

```rust
pub async fn get(
    &self,
    client_id: impl AsRef<str>,
    installation_id: impl AsRef<str>,
    scope: impl Into<MeasurementScope>,
    from: Option<impl DateInput>,
    to: Option<impl DateInput>,
) -> Result<Vec<Measurement>, EnergiaProError>
```

<ParamField path="client_id" type="impl AsRef<str>" required>
  The client identifier.
</ParamField>

<ParamField path="installation_id" type="impl AsRef<str>" required>
  The installation identifier.
</ParamField>

<ParamField path="scope" type="impl Into<MeasurementScope>" required>
  Measurement scope (e.g., `MeasurementScope::LpnJson`).
</ParamField>

<ParamField path="from" type="Option<impl DateInput>">
  Optional start date for filtering measurements.
</ParamField>

<ParamField path="to" type="Option<impl DateInput>">
  Optional end date for filtering measurements.
</ParamField>

<ResponseField name="Result" type="Result<Vec<Measurement>, EnergiaProError>">
  Returns a vector of `Measurement` objects on success.
</ResponseField>

#### Example

```rust
use energiapro::{EnergiaPro, MeasurementScope};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = EnergiaPro::new("username", "secret_key")?;
    
    let measurements = client.measurements.get(
        "client_id",
        "installation_id",
        MeasurementScope::LpnJson,
        Some("2024-01-01"),
        Some("2024-01-31")
    ).await?;
    
    Ok(())
}
```

### all

Retrieve all measurements for a given installation.

```rust
pub async fn all(
    &self,
    client_id: impl AsRef<str>,
    installation_id: impl AsRef<str>,
    scope: impl Into<MeasurementScope>,
) -> Result<Vec<Measurement>, EnergiaProError>
```

<ParamField path="client_id" type="impl AsRef<str>" required>
  The client identifier.
</ParamField>

<ParamField path="installation_id" type="impl AsRef<str>" required>
  The installation identifier.
</ParamField>

<ParamField path="scope" type="impl Into<MeasurementScope>" required>
  Measurement scope (e.g., `MeasurementScope::LpnJson`).
</ParamField>

<ResponseField name="Result" type="Result<Vec<Measurement>, EnergiaProError>">
  Returns a vector of all `Measurement` objects for the installation.
</ResponseField>

<Note>
  This method retrieves all measurements for the specified installation, which may result in a large amount of data being returned. Consider using the other methods in this resource to filter measurements by date or date range if you do not need the entire dataset.
</Note>

#### Example

```rust
use energiapro::{EnergiaPro, MeasurementScope};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = EnergiaPro::new("username", "secret_key")?;
    
    let measurements = client.measurements.all(
        "client_id",
        "installation_id",
        MeasurementScope::LpnJson
    ).await?;
    
    println!("Retrieved {} measurements", measurements.len());
    
    Ok(())
}
```
