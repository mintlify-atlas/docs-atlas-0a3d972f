## Overview

The `ClientOptions` struct allows you to customize the networking behavior of the EnergiaPro client, including the base URL and request timeout.

## Fields

<ResponseField name="base_url" type="String">
  Base URL for requests to the EnergiaPro API.
  
  **Default:** `"https://web2.holdigaz.ch/espace-client-api/api"`
</ResponseField>

<ResponseField name="timeout" type="Duration">
  Timeout for requests to the EnergiaPro API.
  
  **Default:** `Duration::from_secs(30)` (30 seconds)
</ResponseField>

## Methods

### with_timeout

```rust

```

<ParamField path="timeout" type="Duration" required>

## Complete Example

```rust
use energiapro_sdk::{EnergiaPro, ClientOptions};
use std::time::Duration;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Create custom options with 60-second timeout
    let options = ClientOptions::default()
        .with_timeout(Duration::from_secs(60));
    
    // Create client with custom options
    let client = EnergiaPro::with_options(
        "username",
        "secret_key",
        options
    )?;
    
    // Use the client...
    let installations = client.installations.list("client_id").await?;
    
    Ok(())
}
```
